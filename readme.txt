Ball Physics
A 2D ball physics simulation built with C++ and SFML 3.
Features

Ball spawning and rendering
Velocity-based movement with gravity
Ball-to-ball collision detection and response (elastic collisions)

Requirements

Visual Studio 2022
SFML 3 https://www.sfml-dev.org/download/sfml/3.0.2/
C++17 or later

Setup

Clone the repo
Open BALLPHYSICS.slnx in Visual Studio
Make sure SFML 3 include/lib paths are configured in project properties
Copy SFML DLLs to the output directory
Build and run

Controls

Left Click — Spawn a ball
Right Click — Delete a ball
Mouse Scroll — Increase/Decrease number of balls to spawn
Close window — Exit
